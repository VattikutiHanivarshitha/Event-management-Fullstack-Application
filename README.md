#  Event Management System

This project is a full-stack Event Management Application that allows users to create, manage, and participate in events. It provides a complete platform for organizing events with user-friendly interfaces and efficient backend services.

---

##  Features

-  User Authentication (Login/Register)
-  Create and manage events
-  Event registration system
-  View and join events
-  Admin/Organizer controls
-  Search and filter events
-  Responsive UI

##  Installation & Setup

### 1️ Clone the Repository
git clone https://github.com/your-username/event-management-system.git
cd Event-management-Fullstack-Application-main

---

### 2️ Setup Frontend
cd frontend
npm install
npm start

---

### 3️ Setup Backend
cd backend
npm install
npm start

# OR (if Spring Boot)
mvn spring-boot:run

---

### 4️ Database Setup
- Configure DB connection
- Run required SQL scripts

---

##  How It Works

1. User registers and logs in  
2. Organizer creates events  
3. Users browse available events  
4. Users register for events  
5. Admin manages event data
